# File Upload System

A MERN stack project for uploading single and multiple files with progress tracking.

## Features

- Single file upload
- Multiple file upload
- Progress bar
- Upload status tracking

## Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB

## How to Run

### Frontend

```bash
cd client
npm install
npm start
```

### Backend

```bash
cd server
npm install
npm start
```
